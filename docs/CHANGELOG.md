# Changelog

Все заметки о изменениях будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-05-19

### Добавлено
- Базовый MCP-фреймворк (QMCPServer) с поддержкой JSON-RPC 2.0
- Модуль работы с OpenRouter API
- Инструменты: `list_accounts`, `add_account`, `remove_account`, `get_paid_models`, `get_free_models`, `send_chat_request`
- Система аккаунтов с шифрованием API-ключей
- Логгер с ротацией файлов
- Поддержка прокси-сервера
- Компиляция под Qt 5.15.2 и Qt 6.x

### Безопасность
- API-ключи никогда не возвращаются клиенту
- Путь к файлу аккаунтов скрывается
- Автоматическая маскировка чувствительных данных в логах

### Совместимость
- Qt 5.9.9 - поддерживается (с ограничениями)
- Qt 5.15.2 - полная поддержка
- Qt 6.5.1 - полная поддержка
- Qt 6.7.2 - полная поддержка
- Qt 6.10.2 - полная поддержка

### Структура
- Перенесена в стандартную структуру открытого проекта
- Добавлены директории: `include/`, `src/`, `docs/`
- Добавлены: `.gitignore`, `LICENSE`, `CHANGELOG.md`
