# Архитектура OpenRouter MCP Server

## Обзор

OpenRouter MCP Server - это сервер на C++/Qt, реализующий JSON-RPC 2.0 протокол для взаимодействия с OpenRouter API и совместимыми сервисами.

## Компоненты

### 1. QMCPServer (Базовый класс)
**Назначение:** Ядро MCP-фреймворка

**Функции:**
- Обработка JSON-RPC 2.0 сообщений через stdin/stdout
- Маршрутизация вызовов к инструментам
- Управление жизненным циклом сервера
- Базовая обработка ошибок

**Файлы:**
- `include/QMCPServer.h` / `src/QMCPServer.cpp`

### 2. OpenRouterMCPServer (Сервер приложений)
**Назначение:** Реализация MCP-инструментов для OpenRouter

**Инструменты:**
| Инструмент | Описание |
|---|---|
| `list_accounts` | Получить список аккаунтов |
| `add_account` | Добавить новый аккаунт |
| `remove_account` | Удалить аккаунт |
| `get_paid_models` | Список платных моделей |
| `get_free_models` | Список бесплатных моделей |
| `send_chat_request` | Отправить chat-запрос |

**Файлы:**
- `include/OpenRouterMCPServer.h` / `src/OpenRouterMCPServer.cpp`

### 3. McpLogger (Логгер)
**Назначение:** Система логирования

**Функции:**
- Запись в файл с ротацией
- Вывод в stderr
- Маскировка чувствительных данных
- Уровни логирования (Debug, Info, Warn, Error)

**Файлы:**
- `include/McpLogger.h` / `src/McpLogger.cpp`

### 4. ServerConfig (Конфигурация)
**Назначение:** Управление настройками сервера

**Функции:**
- Чтение конфигурации из JSON
- Прокси-сервер настройки
- Управление поведением сервера

**Файлы:**
- `include/ServerConfig.h`

## Структура сообщений

### Входящие (от клиента)
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "tools/call",
  "params": {
    "name": "send_chat_request",
    "arguments": { ... }
  }
}
```

### Исходящие (ответ)
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": "Ответ модели..."
  }
}
```

## Безопасность

1. **API-ключи** никогда не попадают в ответы клиенту
2. **Путь к файлу аккаунтов** скрывается от клиента
3. **Файл accounts.json** должен иметь права доступа 600
4. **Логи** автоматически маскируют чувствительные данные

## Компиляция

```bash
qmake openrouter_mcp.pro
make -j$(nproc)
```

**Требования:**
- Qt 5.15+ или Qt 6.x
- C++17
- Модули: Core, Network
